EZWebGPU.js

EzWebGPU is a wrapper for WebGPU which handles setting up a read and write buffer pair to quickly write, deploy, and visually debug compute shaders very painlessly. You can quickly setup cellular automata simulations and get BILLIONS of FLOPS all in the browser. See on: GitHub
